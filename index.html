Introduction

Introduction

This project is a collection of tools to make it easier to build React Native libraries. It contains the following CLIs:

  • create-react-native-library - a CLI to scaffold a React Native library.
  • react-native-builder-bob - a CLI to build React Native libraries for various targets.
